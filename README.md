# Claims Data Analysis (SQL)

This project uses SQL to analyze a claims dataset and answer business questions related to claim cost, status, and performance.

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite
- CSV dataset

## Key Queries
- Total claim cost by type
- Claim status breakdown
- Average processing time by claim type
- Claim volume by region
- Approval rate calculation

## Key Insights
- Identified highest cost claim categories driving total expenses  
- Analyzed approval vs denial rates to evaluate claim outcomes  
- Evaluated processing times to identify potential inefficiencies  
- Compared claim volume across regions
